# Nigeria Quiz App

A dynamic quiz application testing knowledge about Nigeria's history, government, and current affairs.

## Description

This interactive quiz application randomly selects questions from a pool of Nigerian current affairs topics. The app features:

- Random question selection
- Interactive UI with immediate feedback
- Progress tracking
- Score calculation
- Responsive design
- Result summary with performance messages

## Features

- **Dynamic Question Pool**: Over 30 questions about Nigeria
- **Random Selection**: Each quiz presents 6 random questions
- **Score Tracking**: Real-time score updates
- **Progress Bar**: Visual indication of quiz progress
- **Immediate Feedback**: Color-coded answers (green for correct, red for incorrect)
- **Responsive Design**: Works on both desktop and mobile devices

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/nigeria-quiz-app.git
```

2. Navigate to the project directory:
```bash
cd nigeria-quiz-app
```

3. Open with a local server (e.g., using XAMPP):
- Place the files in your XAMPP's `htdocs` directory
- Start Apache server
- Visit `http://localhost/quizapp/`

## Usage

1. Click "Start Quiz" to begin
2. Read each question carefully
3. Select your answer from the given options
4. See immediate feedback on your selection
5. View your final score and performance message at the end
6. Click "Restart Quiz" to try again with new random questions
